# O que é uma versão de prova?

Os recursos de revisão do [!DNL Workfront’s] ajudam a organização a gerenciar o processo de revisão e aprovação em várias iterações de um arquivo usando versões.

Uma definição básica de versão é uma cópia revisada de uma prova. Por exemplo, o design do estande da organização na conferência de usuários passou pelo processo inicial de revisão e aprovação usando recursos de revisão do [!DNL Workfront’s]. Foram feitas correções e o arquivo atualizado está pronto para revisão.

O novo arquivo pode ser adicionado a [!DNL Workfront] como uma versão do arquivo original.

Isso “empilha” a versão sobre o arquivo original, mantendo um histórico das revisões e garantindo que você esteja visualizando a iteração mais recente do trabalho. O visualizador de revisão do [!DNL Workfront’s] também permite comparar a versão mais recente com uma versão anterior do arquivo.

Vamos começar a revisar e comparar versões de prova.
