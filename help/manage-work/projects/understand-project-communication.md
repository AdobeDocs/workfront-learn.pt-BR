# Entenda a comunicação no projeto

Neste vídeo, você aprenderá a se comunicar com todas as partes interessadas do projeto usando:

* Bom planejamento de projeto
* Atualizações
* Status da tarefa e percentual concluído
* Problemas
* Relatórios
